#include <CGAL/Shape_detection/Region_growing/Polyline/Least_squares_line_fit_region.h>

Definition

Region type based on the quality of the least squares line fit applied to polyline vertices.

This class fits a line, using PCA, to chunks of polyline vertices and controls the quality of this fit. If all quality conditions are satisfied, the chunk is accepted as a valid region, otherwise rejected.

Template Parameters
GeomTraitsa model of Kernel
InputRangea model of ConstRange whose iterator type is RandomAccessIterator
PointMapa model of ReadablePropertyMap whose key type is the value type of the input range and value type is Kernel::Point_2 or Kernel::Point_3
Is Model Of:
RegionType

#include <CGAL/Shape_detection/Region_growing/Polyline/Least_squares_line_fit_region.h>

Definition

Region type based on the quality of the least squares line fit applied to polyline vertices.

This class fits a line, using PCA, to chunks of polyline vertices and controls the quality of this fit. If all quality conditions are satisfied, the chunk is accepted as a valid region, otherwise rejected.

Template Parameters
GeomTraitsa model of Kernel
InputRangea model of ConstRange whose iterator type is RandomAccessIterator
PointMapa model of ReadablePropertyMap whose key type is the value type of the input range and value type is Kernel::Point_2 or Kernel::Point_3
Is Model Of:
RegionType
