Definition

template<typename K>
class CGAL::Interpolation_gradient_fitting_traits_2< K >

Interpolation_gradient_fitting_traits_2 is a model for the concepts InterpolationTraits and GradientFittingTraits.

It can be used to instantiate the geometric traits class of interpolation functions and of Sibson's gradient fitting function when applied on a function defined over a two-dimensional domain. The traits class is templated by a kernel class K.
