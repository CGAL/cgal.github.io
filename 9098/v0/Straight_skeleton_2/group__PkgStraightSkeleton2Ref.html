Reference Manual

Fernando Cacciola, Sébastien Loriot, and Mael Rouxel-Labbé
This package implements an algorithm to construct a halfedge data structure representing the weighted straight skeleton in the interior of 2D polygons with holes and an algorithm to construct inward offset polygons at any offset distance given a straight skeleton.
Introduced in: CGAL 3.2
Depends on: Halfedge Data Structures
BibTeX: cgal:c-sspo2-25b
License: GPL
