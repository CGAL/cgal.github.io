template<typename InputKernel, typename ExactKernel = Exact_predicates_exact_constructions_kernel, typename BaseTraits = Arr_segment_traits_2<ExactKernel>>
struct CGAL::Double_grid_snap_rounding_traits_2< InputKernel, ExactKernel, BaseTraits >

The class Double_grid_snap_rounding_traits_2<InputKernel, ExactKernel, BaseTraits> is a model of the VerticalSlabSnapRoundingTraits_2 concept.

It provides the traits required to perform snap rounding where the resulting points are represented using double precision floating-point coordinates.

Template Parameters
InputKernelspecifies the kernel used for the input geometric objects. It must model the CGAL Kernel concept. These objects must be convertible to the types defined by ExactKernel.
ExactKernelspecifies the exact kernel used internally to perform robust geometric computations. By default, it is set to CGAL::Exact_predicates_exact_constructions_kernel. Advanced users may provide another exact kernel that models the CGAL Kernel concept, such as Cartesian<CGAL::Exact_rational>.
BaseTraitsspecifies the underlying arrangement traits class used for segment handling. By default, it is Arr_segment_traits_2<ExactKernel>. Advanced users may provide another traits that models Aostraits_2 concept.
