Definition

template<typename K, typename ET, typename NT>
class CGAL::Polytope_distance_d_traits_3< K, ET, NT >

The class Polytope_distance_d_traits_3 is a traits class for the \( d\)-dimensional optimization algorithms using the three-dimensional CGAL kernel.

Template Parameters
Kmust be a model for Kernel.
ETmust be models for RingNumberType. The default is K::RT.
