Definition

template<class Traits>
class CGAL::Delaunay_triangulation_on_hyperbolic_surface_2< Traits >

represents a Delaunay triangulation of a closed orientable hyperbolic surface.

The class provides functions such as the generation of the Delaunay triangulation from a convex fundamental domain, the location of a point in the lifted triangulation, the insertion of a point in the Delaunay triangulation, and the \( \varepsilon \)-net algorithm.

Tries to compute an epsilon-net of the surface.

When Number is CGAL::Gmpq, the algorithm rounds the coordinate of circumcenters to CGAL::Gmpfr with precision p*53. When Number is not CGAL::Gmpq, p is ignored.

Precondition
is_epsilon_packing(epsilon) and p > 0

Anchor const &  anch

Returns
a Locate_type that indicates whether query lies on a vertex, an edge, inside or outside the lifted triangle described by the given anchor.

If query lies on a vertex or an edge, the index li is set to the index of the vertex or edge on which query lies.

If query lies inside the triangle, li is set to -1.

If query lies outside the triangle, li is set to the index of the first edge such that query and the third point of the triangle lies on different sides.
