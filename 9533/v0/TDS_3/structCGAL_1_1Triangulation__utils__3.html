◆ next_around_edge()

unsigned int CGAL::Triangulation_utils_3::next_around_edge ( unsigned int i,
unsigned int j )
static

In dimension 3, index of the neighbor n that is next to the current cell, when turning positively around an oriented edge whose endpoints are indexed i and j.

According to the usual numbering of vertices and neighbors in a given cell, it is also the index of the vertex opposite to this neighbor n. (see Figure 49.1).

Precondition
( i < 4 ) && ( j < 4 ) && ( i != j ).
