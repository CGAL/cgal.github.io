Definition

template<class K>
class CGAL::Hyperbolic_Delaunay_triangulation_CK_traits_2< K >

The class Hyperbolic_Delaunay_triangulation_CK_traits_2 is designed as one of the default models for the traits concept HyperbolicDelaunayTriangulationTraits_2 offered by CGAL.

Template Parameters
Kmust be a model of CircularKernel.

This class provides exact constructions and predicates. The default value for K is CGAL::Circular_kernel_2< CGAL::Exact_predicates_inexact_constructions_kernel, CGAL::Algebraic_kernel_for_circles_2_2< CGAL::Exact_predicates_inexact_constructions_kernel::RT > >, which guarantees exact constructions of Delaunay triangulations and dual objects when the input points have rational coordinates.
