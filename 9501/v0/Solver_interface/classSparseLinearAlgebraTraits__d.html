const Vector & B,
Vector & X,
NT & D )

Solve the sparse linear system \(A \times X = B\).

Return true on success. The solution is then \((1/D) \times X \).

Precondition
A.row_dimension() == B.dimension()
A.column_dimension() == X.dimension()
