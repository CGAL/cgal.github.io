VerticalSlabSnapRoundingTraits_2 Concept Reference
Reference Manual » Concepts

Definition

The concept VerticalSlabSnapRoundingTraits_2 lists the set of requirements that must be fulfilled by an instance of the geom_traits parameter of the free function CGAL::vertical_slab_snap_rounding_2() and the functions derived from it. The list includes the nested types of the geometric primitives used in this class and some function object types for the required predicates on those primitives.
