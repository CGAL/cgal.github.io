CGAL::Kinetic_shape_reconstruction_3< Traits, NormalMap > Class Template Reference
Kinetic Partitioning

#include <CGAL/Kinetic_shape_reconstruction_3.h>

Definition

Piece-wise linear reconstruction via inside/outside labeling of a kinetic partition using graph cut.

Template Parameters
GeomTraitsmust be a model of Kinetic_shape_partition_trais_3.
NormalMapmust be a model of ConstRange whose iterator type is RandomAccessIterator.
Examples:
Kinetic_shape_reconstruction/kinetic_reconstruction.cpp.

Provides the data and regularity energy terms for reconstruction via graph-cut.

Parameters
edgescontains a vector of pairs of volume indices. Indicates which volumes should be connected in the graph cut formulation.
edge_costscontains the cost for each edge specified in edges for two labels with different labels. For equal labels, the cost is 0. Needs to be index compatible to the edges parameter.
cost_matrixprovides the cost of a label for each volume cell. The first index corresponds to the label and the second index corresponds to the volume index.
Returns
fails if the dimensions of parameters does not match the kinetic partitioning.
Precondition
successful initialization
