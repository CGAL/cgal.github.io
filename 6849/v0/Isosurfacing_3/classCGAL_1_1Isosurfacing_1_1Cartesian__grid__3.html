Definition

template<typename GeomTraits>
class CGAL::Isosurfacing::Cartesian_grid_3< GeomTraits >

stores scalar values and gradients at the vertices of a Cartesian grid.

Template Parameters
GeomTraitsmust be a model of IsosurfacingTraits_3.
Examples
Isosurfacing_3/all_Cartesian_cube.cpp, Isosurfacing_3/dual_contouring_Cartesian_grid.cpp, Isosurfacing_3/marching_cubes_Cartesian_grid_sphere.cpp, Isosurfacing_3/marching_cubes_inrimage.cpp, and Isosurfacing_3/marching_cubes_signed_mesh_offset.cpp.

creates a grid with xdim * ydim * zdim grid vertices.

The grid covers the space described by a bounding box.

Parameters
xdimthe number of grid vertices in the x direction
ydimthe number of grid vertices in the y direction
zdimthe number of grid vertices in the z direction
bboxthe bounding box
gtthe geometric traits
Precondition
xdim, ydim, and zdim are (strictly) positive.
