#include <CGAL/Isosurfacing_3/Explicit_Cartesian_grid_domain_3.h>

Creates a domain that can be used as input for isosurfacing algorithms.

Warning
The domain keeps a pointer to the grid object, hence users must ensure that the lifetime of the grid object exceeds that of the object returned by this function.
Template Parameters
GeomTraitsmust be a model of IsosurfacingTraits_3.
Gradientthe type of the gradient functor. It must be a model of CopyConstructible and implement GeomTraits::Vector_3 operator()(const GeomTraits::Point_3& point) const.
Parameters
gridthe Cartesian grid containing input data
gradienta function giving the value of the gradient of the implicit function at each discretization point

a new CGAL::Isosurfacing::Explicit_Cartesian_grid_domain_3

#include <CGAL/Isosurfacing_3/Implicit_Cartesian_grid_domain_3.h>

creates a domain from an implicit function that can be used as input for isosurfacing algorithms.

The implicit function is evaluated at the vertices of the virtual grid defined by the bounding box and the spacing value. By not storing any function values explicitely, less overall memory is required in comparison to an Explicit_Cartesian_grid_domain_3.

Template Parameters
GeomTraitsmust be a model of IsosurfacingTraits_3.
ImplicitFunctionthe type of the implicit function. It must be a model of CopyConstructible and implement GeomTraits::FT operator()(const GeomTraits::Point_3& point) const.
Gradientthe type of the gradient functor. It must be a model of CopyConstructible and implement GeomTraits::Vector_3 operator()(const GeomTraits::Point_3& point) const.
Parameters
bboxa bounding box that specifies the dimensions of the implicit function's domain
spacingthe distance between discretization points
point_functionthe implicit function giving the value of the implicit function at each discretization point
gradienta function giving the value of the gradient of the implicit function at each discretization point
gtan instance of geometric traits
Returns
a new object of type CGAL::isosurfacing::Implicit_Cartesian_grid_domain_3
Precondition
spacing != CGAL::NULL_VECTOR
