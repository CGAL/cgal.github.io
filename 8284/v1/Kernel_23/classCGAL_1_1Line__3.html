#include <CGAL/Line_3.h>

Definition

template<typename Kernel>
class CGAL::Line_3< Kernel >

An object l of the data type Line_3 is a directed straight line in the three-dimensional Euclidean space \( \E^3\).

Is model of
Kernel::Line_3
