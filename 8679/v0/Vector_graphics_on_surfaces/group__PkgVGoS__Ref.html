Vector Graphics on Triangulated Surface Meshes Reference

Claudio Mancinelli and Sébastien Loriot
The package provides functions to draw and manipulate basic geometric primitives such as geodesics curves and Bézier splines on triangulated surface meshes.
Introduced in: CGAL 6.2
Depends on: Polygon Mesh Processing
BibTeX: cgal:ml-vgos-25a
License: GPL
Windows Demo: CGAL Lab
