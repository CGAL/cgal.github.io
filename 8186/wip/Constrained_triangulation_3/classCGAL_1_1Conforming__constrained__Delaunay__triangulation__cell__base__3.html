template<typename Traits, typename Cell_base = Triangulation_cell_base_3<Traits>>
class CGAL::Conforming_constrained_Delaunay_triangulation_cell_base_3< Traits, Cell_base >

Cell base class for the 3D conforming constrained Delaunay triangulation.

This class is derived from the Triangulation_cell_base_3 class and provides additional functionality required by make_conforming_constrained_Delaunay_triangulation_3().

Template Parameters
TraitsThe geometric traits class, which must be a model of ConformingConstrainedDelaunayTriangulationTraits_3. It should be the same as the geometric traits class of the triangulation.
Cell_baseThe base class for the cell, which must be a model of TriangulationCellBase_3.
Is model of
ConformingConstrainedDelaunayTriangulationCellBase_3
SimplicialMeshCellBase_3
RemeshingCellBase_3
Note
This cell base class also models the SimplicialMeshCellBase_3 and RemeshingCellBase_3 concepts, allowing the use of functionality from Tetrahedral Remeshing and 3D Simplicial Mesh Data Structures, if the corresponding vertex base also models the right concepts.
