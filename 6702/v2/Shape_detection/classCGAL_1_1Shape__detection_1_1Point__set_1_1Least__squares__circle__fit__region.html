#include <CGAL/Shape_detection/Region_growing/Point_set/Least_squares_circle_fit_region.h>

Definition

Region type based on the quality of the least squares circle fit applied to 2D points.

This class fits a circle to chunks of points in a 2D point set and controls the quality of this fit. If all quality conditions are satisfied, the chunk is accepted as a valid region, otherwise rejected.

Template Parameters
GeomTraitsa model of Kernel
InputRangea model of ConstRange whose iterator type is RandomAccessIterator
PointMapa model of ReadablePropertyMap whose key type is the value type of the input range and value type is Kernel::Point_2
NormalMapa model of ReadablePropertyMap whose key type is the value type of the input range and value type is Kernel::Vector_2

class CGAL::Shape_detection::Point_set::Least_squares_circle_fit_region< GeomTraits, InputRange, PointMap, NormalMap >

Region type based on the quality of the least squares circle fit applied to 2D points.

This class fits a circle to chunks of points in a 2D point set and controls the quality of this fit. If all quality conditions are satisfied, the chunk is accepted as a valid region, otherwise rejected.
