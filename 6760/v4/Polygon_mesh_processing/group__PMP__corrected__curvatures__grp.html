#include <CGAL/Polygon_mesh_processing/Curvatures/interpolated_corrected_curvature_measures.h>

Expands given face area and anisotropic measures around a vertex v.

Expansion is based on the inclusion ratio of each face in a ball of radius r around the vertex v.

Template Parameters
PolygonMesha model of FaceListGraph
AreaFaceMeasureMapa model of WritablePropertyMap with boost::graph_traits<PolygonMesh>::face_descriptor as key type and GT::FT as value type.
AnisotropicFaceMeasureMapa model of WritablePropertyMap with boost::graph_traits<PolygonMesh>::face_descriptor as key type and std::array<typename GT::FT, 3 * 3> as value type.
AreaVertexMeasureMapa model of WritablePropertyMap with boost::graph_traits<PolygonMesh>::vertex_descriptor as key type and GT::FT as value type.
AnisotropicVertexMeasureMapa model of WritablePropertyMap with boost::graph_traits<PolygonMesh>::vertex_descriptor as key type and std::array<typename GT::FT, 3 * 3> as value type.
NamedParametersa sequence of Named Parameters.
Parameters
pmeshthe polygon mesh.
area_fmm(area face measure map) the property map storing the already computed area measure on each face.
aniso_fmm(anisotropic face measure map) the property map storing the already computed anisotropic measure on each face.
area_vmm(area vertex measure map) the property map provided to store the expanded area measure on each vertex.
aniso_vmm(anisotropic vertex measure map) the property map provided to store the expanded anisotropic measure on each vertex.
v(vertex) the vertex to expand the area and anisotropic measure around.
npan optional sequence of Named Parameters among the ones listed below
Optional Named Parameters
  • the radius of the ball around the vertex v to expand the area and curvature measure
  • Type: GT::FT
  • Default: 0.01

  • a property map associating points to the vertices of pmesh
  • Type: a class model of ReadablePropertyMap with boost::graph_traits<PolygonMesh>::vertex_descriptor as key type and Point_3 as value type
  • Default: boost::get(CGAL::vertex_point, pmesh)
  • Extra: If this parameter is omitted, an internal property map for CGAL::vertex_point_t must be available in PolygonMesh.

#include <CGAL/Polygon_mesh_processing/Curvatures/interpolated_corrected_curvature_measures.h>

Expands given face area and curvature (mean or gaussian) measures around a vertex v.

Expansion is based on the inclusion ratio of each face in a ball of radius r around the vertex v.

Template Parameters
PolygonMesha model of FaceListGraph.
FaceMeasureMapa model of WritablePropertyMap with boost::graph_traits<PolygonMesh>::face_descriptor as key type and GT::FT as value type.
VertexMeasureMapa model of WritablePropertyMap with boost::graph_traits<PolygonMesh>::vertex_descriptor as key type and GT::FT as value type.
NamedParametersa sequence of Named Parameters.
Parameters
pmeshthe polygon mesh.
area_fmm(area face measure map) the property map storing the already computed area measure on each face.
curvature_fmm(curvature face measure map) the property map storing the already computed curvature measure on each face. This curvature measure can be either the Mean Curvature or the Gaussian Curvature.
area_vmm(area vertex measure map) the property map provided to store the expanded area measure on each vertex.
curvature_vmm(curvature vertex measure map) the property map provided to store the expanded curvature measure on each vertex. This curvature measure can be either the Mean Curvature or the Gaussian Curvature.
v(vertex) the vertex to expand the area and curvature measure around.
npan optional sequence of Named Parameters among the ones listed below
Optional Named Parameters
  • the radius of the ball around the vertex v to expand the area and curvature measure
  • Type: GT::FT
  • Default: 0.01

  • a property map associating points to the vertices of pmesh
  • Type: a class model of ReadablePropertyMap with boost::graph_traits<PolygonMesh>::vertex_descriptor as key type and Point_3 as value type
  • Default: boost::get(CGAL::vertex_point, pmesh)
  • Extra: If this parameter is omitted, an internal property map for CGAL::vertex_point_t must be available in PolygonMesh.

Computes the interpolated corrected anisotropic measure on each face of the mesh.

Template Parameters
PolygonMesha model of FaceListGraph.
FaceMeasureMapa model of WritablePropertyMap with boost::graph_traits<PolygonMesh>::face_descriptor as key type and std::array<GT::FT, 3 * 3> as value type.
NamedParametersa sequence of Named Parameters.
Parameters
pmeshthe polygon mesh.
fmm(face measure map) the property map used for storing the computed interpolated corrected measure.
npan optional sequence of Named Parameters among the ones listed below
Optional Named Parameters
  • a property map associating points to the vertices of pmesh
  • Type: a class model of ReadablePropertyMap with boost::graph_traits<PolygonMesh>::vertex_descriptor as key type and Point_3 as value type
  • Default: boost::get(CGAL::vertex_point, pmesh)
  • Extra: If this parameter is omitted, an internal property map for CGAL::vertex_point_t must be available in PolygonMesh.

  • a property map associating normal vectors to the vertices of pmesh
  • Type: a class model of ReadablePropertyMap with boost::graph_traits<PolygonMesh>::vertex_descriptor as key type and Vector_3 as value type
  • Default: get(dynamic_vertex_property_t<GT::Vector_3>(), pmesh)
  • Extra: If this parameter is omitted, vertex normals will be computed using compute_vertex_normals()

Expansion is based on the inclusion ratio of each face in a ball of radius r around the vertex v.
