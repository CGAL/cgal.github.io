template<typename InputRange , typename OutputIterator , typename CGAL_NP_TEMPLATE_PARAMETERS >
OutputIterator CGAL::Shape_detection::region_growing_lines ( const InputRange &  points,
OutputIterator  regions,
const CGAL_NP_CLASS &  np = parameters::default_values() 
)

#include <CGAL/Shape_detection/Region_growing/free_functions.h>

helper function to facilitate region growing for line detection on point sets.

Template Parameters
InputRangea model of ConstRange
OutputIteratora model of OutputIterator accepting a std::pair<Kernel::Line_2, std::vector<InputRange::const_iterator> > or std::pair<Kernel::Line_3, std::vector<InputRange::const_iterator> >.
NamedParametersa sequence of Named Parameters
Parameters
pointsinput point range for region growing.
regionsoutput iterator to store the detected regions and fitted lines.
npan optional sequence of Named Parameters among the ones listed below
Optional Named Parameters
  • the number of returned neighbors per each query point
  • Type: std::size_t
  • Default: 12
  • the maximum distance from a point to a line
  • Type: GeomTraits::FT
  • Default: 1
  • the maximum angle in degrees between the normal of a point and the normal of a line
  • Type: GeomTraits::FT
  • Default: 25 degrees
  • the cos value computed as cos(maximum_angle * PI / 180), this parameter can be used instead of the maximum_angle
  • Type: GeomTraits::FT
  • Default: cos(25 * PI / 180)
  • the minimum number of 2D points a region must have
  • Type: std::size_t
  • Default: 2
  • an instance of PointMap that maps an item from input_range to Kernel::Point_2
  • Default: PointMap()
  • an instance of NormalMap that maps an item from input_range to Kernel::Vector_2
  • Default: NormalMap()
  • an instance of GeomTraits
  • Default: GeomTraits()
