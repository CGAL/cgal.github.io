ConstrainedTriangulationTraits_2 Concept Reference
2D Triangulations Reference » Concepts

Definition

The concept ConstrainedTriangulationTraits_2 defines the requirements for the geometric traits class of a constrained triangulation ( CGAL::Constrained_triangulation_2<Traits,Tds,Itag>) that supports intersections of input constraints (i. e. when the template parameter Itag is instantiated by one of the tag classes CGAL::Exact_intersections_tag or CGAL::Exact_predicates_tag). This concept refines the concept TriangulationTraits_2, adding requirements for function objects to compute the intersection points of two constraints. When CGAL::Exact_predicates_tag is used, the traits class is also required to provide additional types to compute the squared distance between a point and a line.

Types required with Exact_predicates_tag

When the constrained triangulation is instantiated with the intersection tag CGAL::Exact_predicates_tag, the used algorithm needs to be able to compare some distances between points and lines and the following types are further required.
