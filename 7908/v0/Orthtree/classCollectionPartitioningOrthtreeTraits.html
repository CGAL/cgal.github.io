CollectionPartitioningOrthtreeTraits Concept Reference
Quadtree\, Octree and Orthtree Reference » Concepts

Definition

In addition to the requirements described in the OrthtreeTraits concept, the concept CollectionPartitioningOrthtreeTraits defines the requirements for the traits class of a CGAL::Orthtree which supports nearest-neighbor searching.

Nearest neighbor searches expect a tree with nodes which contain list types. The leaf nodes of the tree represent an exclusive partition of the elements contained in the tree. This means that no element should be contained by more than one node.
