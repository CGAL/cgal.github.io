RectangularPCenterTraits_2 Concept Reference
Reference Manual » Concepts

Definition

The concept RectangularPCenterTraits_2 defines types and operations needed to compute rectilinear \( p\)-centers of a planar point set using the function CGAL::rectangular_p_center_2().

Has models

typedef unspecified_type RectangularPCenterTraits_2::Construct_point_2_above_left_implicit_point_2

3-argument function class: Point_2 \( \times\) Point_2 \( \times\) FT \( \rightarrow\) Point_2.

For arguments \( (p,\,q,\,r)\) it returns the upper-left corner of the iso-oriented square with sidelength \( r\) and lower-right corner at the intersection of the vertical line through \( p\) and the horizontal line through \( q\).
