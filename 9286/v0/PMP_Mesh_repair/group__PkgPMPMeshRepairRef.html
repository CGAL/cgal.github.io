Reference Manual

Sébastien Loriot, Mael Rouxel-Labbé, and Ilker O. Yaz
This CGAL package provides methods that enable treating both combinatorial and geometrical defects in polygon meshes and polygon soups: orientation, hole filling, degeneracies removal, boundary stitching, and more.
Introduced in: CGAL 4.7
Depends on: documented for each function
BibTeX: cgal:pmp-repair-26a
License: GPL
Windows Demo: CGAL Lab
