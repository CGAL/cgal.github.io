#include <CGAL/Bbox_d.h>

Definition

template<typename DimensionTag>
class CGAL::Bbox_d< DimensionTag >

An object b of the class Bbox_d is a bounding box in the d-dimensional Euclidean plane \( \E^d\).

This class is templated with a dimension tag.
