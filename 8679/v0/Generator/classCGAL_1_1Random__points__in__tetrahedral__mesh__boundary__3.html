class CGAL::Random_points_in_tetrahedral_mesh_boundary_3< C3T3, Creator >

The class Random_points_in_tetrahedral_mesh_boundary_3 is an input iterator creating points uniformly distributed on the boundary of a tetrahedral mesh of type Mesh_complex_3_in_triangulation_3.

The tetrahedral mesh must be valid and unchanged while the iterator is used.
