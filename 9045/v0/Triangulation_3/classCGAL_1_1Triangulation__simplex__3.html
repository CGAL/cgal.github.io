Definition

template<typename TriangulationDataStructure_3>
class CGAL::Triangulation_simplex_3< TriangulationDataStructure_3 >

The class Triangulation_simplex_3 stores a simplex of any dimension defined by the TriangulationDataStructure_3 class.

It also defines the operator less such that simplices can be stored in a map or a set of simplices. The simplex is invalidated by any change in the triangulation.

Parameters

It is parameterized by the triangulation it derives the simplices from.
