class CGAL::Constrained_Delaunay_triangulation_vertex_base_3< Gt, Vb >

The Constrained_Delaunay_triangulation_vertex_base_3 class is a vertex base class for the Constrained Delaunay Triangulation in 3D.

This class is derived from the Triangulation_vertex_base_3 class and provides additional functionality required by make_constrained_Delaunay_triangulation_3().

Template Parameters
GtThe geometric traits class, model of DelaunayTriangulationTraits_3. It must be the same as the geometric traits class of the triangulation.
VbThe base class for the vertex. It must be a model of TriangulationVertexBase_3.
