Definition

template<typename MD>
class CGAL::Mesh_domain_with_polyline_features_3< MD >

The class Mesh_domain_with_polyline_features_3 enables the user to add some 0- and 1-dimensional features into any model of the MeshDomain_3 concept.

The 1-dimensional features are described as polylines whose endpoints are the added corners.

Template Parameters
MDis the type of the domain which is extended. It has to be a model of the MeshDomain_3 concept.
