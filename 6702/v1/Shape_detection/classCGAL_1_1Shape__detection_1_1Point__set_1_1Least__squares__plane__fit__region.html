#include <CGAL/Shape_detection/Region_growing/Point_set/Least_squares_plane_fit_region.h>

Definition

Region type based on the quality of the least squares plane fit applied to 3D points.

This class fits a plane, using PCA, to chunks of points in a 3D point set and controls the quality of this fit. If all quality conditions are satisfied, the chunk is accepted as a valid region, otherwise rejected.

Template Parameters
GeomTraitsa model of Kernel
InputRangea model of ConstRange whose iterator type is RandomAccessIterator
PointMapa model of ReadablePropertyMap whose key type is the value type of the input range and value type is Kernel::Point_3
NormalMapa model of ReadablePropertyMap whose key type is the value type of the input range and value type is Kernel::Vector_3
Is Model Of:
RegionType
