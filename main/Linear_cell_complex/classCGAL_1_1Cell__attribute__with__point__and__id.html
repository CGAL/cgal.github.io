Definition

template<typename LCC, typename Info_, typename Tag, typename OnMerge, typename OnSplit>
class CGAL::Cell_attribute_with_point_and_id< LCC, Info_, Tag, OnMerge, OnSplit >

The class Cell_attribute_with_point_and_id represents an attribute containing a point, containing an information when Info_ is different from void, and having an id.

This class can typically be used to associate a point with id to each 0-cell of a combinatorial or a generalized map.

Info_the type of the information contained in the attribute, void for no information. Equal to void by default.
Tagis ::Tag_true to enable the storage of a Dart_descriptor of the associated cell, ::Tag_false otherwise. Equal to ::Tag_true by default.
OnMergea functor called when two attributes are merged. Equal to Null_functor by default.
