3D Isosurfacing Reference

Julian Stahl, Daniel Zint and Pierre Alliez
This package implements several isosurfacing algorithms (Marching Cubes and its topologically correct variant, and Dual Contouring) to generate a surface mesh from a 3D scalar field. The possible format for the input scalar field is flexible and includes explicit as well as implicit representations.
Introduced in: CGAL 5.6
BibTeX: cgal:sz-mc-24a
License: GPL
Windows Demo: Polyhedron demo

This package provides algorithms to extract isosurfaces from different inputs: Marching Cubes, topologically correct Marching Cubes, and Dual Contouring. The input is represented as a 3D scalar field defined as an implicit function, a Cartesian grid or an octree. The output is a surface mesh represented as an indexed face set (container of point coordinates and indexed faces). The provided algorithms include Marching Cubes, topologically correct Marching Cubes and Dual Contouring.
