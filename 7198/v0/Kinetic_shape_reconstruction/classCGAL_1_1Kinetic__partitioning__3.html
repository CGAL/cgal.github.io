template<Kernel, Intersection_Kernel = CGAL::Exact_predicates_exact_constructions_kernel>
class CGAL::Kinetic_partitioning_3< Kernel, Intersection_Kernel >

Creates the kinetic partitioning of the bounding box.

Template Parameters
Kernelmust be a model of Kernel. Is used for non-critical calculations.
IntersectionKernelmust be a model of Kernel. Is used for the creation of the intersection graph. An exact kernel is suggested.

PolygonMapcontains index ranges to form polygons from InputRange
NamedParametersa sequence of Named Parameters
Parameters
input_rangean instance of InputRange with 3D points and corresponding 3D normal vectors
polygon_mapa range of polygons defined by a range of indices into input_range
npa sequence of Named Parameters among the ones listed below
Returns
success
Precondition
input_range.size() > 0 and polygon_map.size() > 0
Optional Named Parameters
  • Use the oriented bounding box instead of the axis-aligned bounding box.
  • Type: bool
  • Default: false
  • Factor for extension of the bounding box of the input data to be used for the partitioning.
  • Type: FT
  • Default: 1.1
  • The tolerance angle to snap the planes of two input polygons into one plane.
  • Type: FT
  • Default: 5
  • The tolerance distance to snap the planes of two input polygons into one plane.

  • Default: 5
