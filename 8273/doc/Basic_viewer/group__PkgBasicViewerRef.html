Basic Viewer Reference

Guillaume Damiand and Mostafa Ashraf
This package provides interactive visualization for most CGAL packages. First, the function draw() is provided for each data structure. Several data structures can be added to a same graphics scene and visualized simultaneously. Rendering can be controlled (color, point size, etc.). Finally a Qt widget is provided that can be integrated in the GUI of a user application.
Introduced in: CGAL 6.0
