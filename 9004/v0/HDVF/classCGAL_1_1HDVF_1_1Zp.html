Definition

template<int p, typename _TSlot = int>
class CGAL::HDVF::Zp< p, _TSlot >

The class Zp implements the concept Ring with the field \(\mathbb Z/p\mathbb Z\).

This is a "lightweight" implementation which aims at providing fast operations and constructors.

According to the value of p, users can chose the size of the representation used to store values (default size: int).

Warning
For \(\mathbb Z/2\mathbb 2\), prefer the class Z2 which is optimized.
