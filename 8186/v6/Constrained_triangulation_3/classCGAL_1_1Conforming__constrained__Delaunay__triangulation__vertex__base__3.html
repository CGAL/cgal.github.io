template<typename Traits, typename VertexBase = Triangulation_vertex_base_3<Traits>>
class CGAL::Conforming_constrained_Delaunay_triangulation_vertex_base_3< Traits, VertexBase >

Vertex base class for the 3D conforming constrained Delaunay triangulation.

This class is derived from the Triangulation_vertex_base_3 class and provides additional functionality required by make_conforming_constrained_Delaunay_triangulation_3().

Template Parameters
TraitsThe geometric traits class, model of ConformingConstrainedDelaunayTriangulationTraits_3. It must be the same as the geometric traits class of the triangulation.
VertexBaseThe base class for the vertex. It must be a model of TriangulationVertexBase_3.
