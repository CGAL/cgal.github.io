PMPSizingField Concept Reference
Reference Manual » Concepts

Definition

The concept PMPSizingField defines the requirements for the sizing field used in CGAL::Polygon_mesh_processing::isotropic_remeshing() to define the target length for every individual edge during the remeshing process.

Has models
CGAL::Polygon_mesh_processing::Uniform_sizing_field
CGAL::Polygon_mesh_processing::Adaptive_sizing_field

Types

These types are used for the documentation of the functions of the concept and not needed implementation wise.
