detects and constructs the feature graph from a labeled image

The feature graph has polylines that lie on the sharp features of the surface defined by the image's subdomains. Each subdomain inside the bounding box of the input labeled image is defined as the set of voxels with the same value. The outside of the bounding box of the image is considered as a subdomain with voxel value specified by parameters::value_outside.
